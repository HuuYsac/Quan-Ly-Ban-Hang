# Phần Mềm Quản Lý Bán Hàng - Hữu Laptop

Đây là hệ thống quản lý bán hàng chuyên nghiệp được tối ưu hóa cho các cửa hàng kinh doanh thiết bị công nghệ, laptop, và linh kiện máy tính. Phần mềm giúp số hóa quy trình quản lý, từ khâu nhập kho, theo dõi đơn hàng đến chăm sóc khách hàng sau bán.

## 🚀 Tính Năng Nổi Bật

- **Quản lý kho hàng thông minh:** Theo dõi chi tiết cấu hình máy (CPU, RAM, SSD, Màn hình), số lượng tồn kho và lịch sử nhập/xuất.
- **Quản lý đơn hàng & Doanh thu:** Tạo hóa đơn nhanh chóng, theo dõi trạng thái thanh toán và báo cáo doanh thu theo thời gian thực.
- **Quản lý khách hàng (CRM):** Lưu trữ thông tin khách hàng, lịch sử mua hàng để hỗ trợ các chương trình hậu mãi và bảo hành.
- **Tối ưu hóa quy trình bảo hành:** Theo dõi thời hạn bảo hành của từng linh kiện, máy tính dựa trên số Serial/Service Tag.
- **Giao diện thân thiện:** Thiết kế tối giản, dễ sử dụng ngay cả với người không rành về kỹ thuật.

## 🛠 Công Nghệ Sử Dụng

- **Phát triển bởi:** Google AI Studio & AppSheet.
- **Nền tảng:** Cloud-based (Truy cập mọi lúc, mọi nơi trên máy tính và điện thoại).
- **Lưu trữ dữ liệu:** Google Sheets / Database chuyên dụng.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
